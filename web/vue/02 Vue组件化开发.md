# Vue 组件化开发



## 命名

#### 组件名

一个单词：

​		首字母小写：school

​		首字母大写：School 

多个单词：

​		kebab-case命名：my-school

​		CamelCase命名：MySchool（需要Vue脚手架支持）

可以使用name配置项指定组件在开发者工具中呈现的名字。



#### 组件标签

`<school></school>`
