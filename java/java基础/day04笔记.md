###04.09_Java语言基础(循环结构三种循环语句的区别)
* A:案例演示
	* 三种循环语句的区别:
	* do...while循环至少执行一次循环体。
	* 而for,while循环必须先判断条件是否成立，然后决定是否执行循环体语句。
* B:案例演示
	* for循环和while循环的区别：
		* A:如果你想在循环结束后，继续使用控制条件的那个变量，用while循环，否则用for循环。不知道用for循环。因为变量及早的从内存中消失，可以提高内存的使用效率。
		* B:建议:
			* 如果是一个范围的，用for循环非常明确。
			* 如果是不明确要做多少次，用while循环较为合适。
				* 举例：吃葡萄。
				* while(x != 0)
